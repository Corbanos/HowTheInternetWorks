1991 - The Web

Tim Berners-Lee introduced the World Wide Web, making the internet widely accessible.

The Web transformed how we access and share information.

What is the World Wide Web?

The World Wide Web (WWW) is a system of interlinked hypertext documents accessed via the internet. It allows users to navigate between documents using hyperlinks.

Key Innovations

  • HTML: Hypertext Markup Language, the standard language for creating web pages.
  • HTTP: Hypertext Transfer Protocol, the protocol used to transfer web pages.
  • Web Browsers: Software applications like Mosaic, Netscape, and modern browsers like Chrome and Firefox that allow users to access the web.

Impact on Society

The Web has revolutionized communication, commerce, education, and entertainment. It has made information more accessible and connected people worldwide.
